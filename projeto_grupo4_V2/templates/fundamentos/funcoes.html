Funções e Procedimentos

Conceito:

Funções são blocos de código reutilizáveis que executam uma tarefa e podem ou não retornar um valor.

Aplicação:

Organizar melhor o código e evitar repetição.

Exemplo:

def saudacao(nome):
    print("Olá,", nome)

saudacao("Lucas")

Com retorno:

def soma(a, b):
    return a + b

resultado = soma(3, 5)
print(resultado)
