Estruturas de Repetição

Conceito:

Permite que um bloco de código seja repetido várias vezes, com base em uma condição ou intervalo.

Aplicação:

Repetir ações como percorrer listas, somar valores ou exibir mensagens.

Exemplo com for:

            for i in range(5)
                print("Contagem:", i)

Com while:

        contador = 0
        while contador < 5:
            print("Contador:", contador)
            contador += 1
